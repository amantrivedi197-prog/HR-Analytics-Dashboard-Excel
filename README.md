# 📊 HR Analytics Dashboard – Microsoft Excel

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard developed using Microsoft Excel to analyze employee data and generate meaningful business insights.

The entire HR dataset was *created manually from scratch* to simulate a real-world HR environment. Instead of using a ready-made dataset, I designed and organized the data myself.

To improve data organization and analysis, I created *multiple related tables* and performed *data modeling* by establishing relationships between them. This approach ensures accurate reporting and makes the dashboard more dynamic and scalable.

The dashboard helps analyze employee demographics, salary distribution, attrition rate, department performance, job roles, locations, and workforce trends through interactive visualizations.

## 🎯 Project Objectives

- Analyze employee workforce statistics.
- Monitor employee attrition.
- Compare average salary across departments.
- Analyze gender distribution.
- Track employee count trends.
